📖 QUICK START
═══════════════════════════════════════════════════════════════

1. Load Extension:
   → chrome://extensions/
   → Developer mode ON
   → Load unpacked → Select DocuGuide folder

2. Test It:
   → Go to Wikipedia
   → Select a paragraph
   → Open DocuGuide sidebar
   → See green bar? You're good!
   → Click "Summarize"

3. If It Works:
   → You're done! Extension is ready
   → See HOW-TO-USE.md for full guide

4. If It Doesn't Work:
   → Check FIXES-APPLIED.md for troubleshooting
   → Make sure you're on Chrome 138+
   → Check chrome://on-device-internals
